## Installation

Cloning this repo and running:

    # make install

### Installing Binaries

Install a few ansible cli tools:

    $ avm 1.9.4
    $ avm 2.0.0.2

Type `avm` to prompt selection of an installed ansible. Use the up /
down arrow to navigate, and press enter or the right arrow to
select, or ^C to cancel:

    $ avm

      ansible/1.9.4
    -> ansible/2.0.0.0
      ansible/2.0.0.2

Use or install the latest official release:

    $ avm latest

### Removing Binaries

Remove some versions:

    $ avm rm 2.0.0.0
